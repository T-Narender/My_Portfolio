# Narender's Portfolio

A modern, responsive personal portfolio website for a software engineering student.

## Features

- Clean, minimal design with navy blue, white, and teal color scheme
- Fully responsive (desktop, tablet, mobile)
- Smooth scrolling navigation with sticky navbar
- Interactive contact form with toast notifications
- Showcase of technical skills and projects
- Professional animations and hover effects

## Technologies Used

- HTML5
- CSS3 (with custom variables and animations)
- Vanilla JavaScript
- Lucide Icons
- Inter Font

## Sections

- **Hero Section**: Introduction with call-to-action buttons
- **About Me**: Personal journey and goals
- **Skills**: Technical skills categorized by domain
- **Projects**: Featured projects with descriptions and tech stacks
- **Contact**: Contact form and social media links

## How to Run

Simply open `index.html` in your web browser. No build process required!
